# Streams Editor

Visual device editor with 19 devices, browser modes, safe zones, click-to-inspect, visual editing, GitHub push.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000/editor

## Deploy to Vercel

1. Create repo `streams-editor` on GitHub
2. Push this code to the repo
3. Import into Vercel
4. Auto-deploys

## Push commands

```bash
git init
git add .
git commit -m "Initial commit - Streams Editor"
git branch -M main
git remote add origin https://github.com/hawk7227/streams-editor.git
git push -u origin main
```
